# Android Modules

### Introduction

The `android` category contains modules focused on Android application inspection and auxiliary APK-related tasks.

### Current modules

- `Android::APKSign`
- `Android::Manifest`
- `Android::Schemes`
- `Android::Strings`

### Typical use cases

- inspect Android manifests
- review package metadata and strings
- enumerate URI schemes
- support APK-related testing workflows

### Notes

- Android modules operate on local artifacts rather than remote targets
- some supporting notes also exist under `lib/Spellbook/Android/notes.txt`
